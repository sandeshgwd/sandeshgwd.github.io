Autonomous delivery system

  • Coolio

    I co-founded 'Coolio', an autonomous in-Airport delivery system for travelers in 2016 as a side gig. Airports see high volume of passengers, food courts spread across; however people have restricted options around the vicinity of the gates. Our goal was to provide travellers and staff access to whatever they want and delivered to their gate.
    I worked on building the platform and developed 5 different prototypes from lazer cut options to 3D modelling. Also implemented a control architecture with hierarchical planner for navigation. This helped the robot maneuver through tight spaces, crowded environment and also detect small obstacles. The videos will give more insight to this.

              

    Thesis

    • Autonomous Robotic Landmine Detection, Mapping and Avoidance

      The thesis put forth the socially challenging issue of Landmine detection and deals with a solution strategy for localization of metals using non-linear optimization, usage of machine learning radial basis support vector machine classifier to distinguish between metal samples including surrogate mines and avoidance of the mines using potential field approach.
      Designed C++ & ROS multi-process based system for the approach.
      Interfaced multiple sensors like LIDAR, camera, infrared sensors and odometry for motion/path planning and worked with multiple robot – youBot, PR2 and single-board odroid.
      Upgraded Labo3a (AAI) diff. drive robot to ROS framework platform, incorporated SLAM gmapping and autonomous navigation using A* algorithm. Worked on low-level device drivers and RTAI kernel patch
      Also worked with MATLAB, CAD design and, git for code maintenance and review.

      • Masters

      • Landmine detection and avoidance - HRATC'15

        Improvised on the Metal detector platform of Humanitarian Robotics and Automation 2014 Challenge (HRATC) with classification - machine learning techniques, optimization - estimation techniques and navigation strategies for surrogate landmine identification and its avoidance.

      • Design of a Preemptive and Cooperative Real-time Operating System

        Implemented selectable cooperative or preemptive RTOS in Embedded C. Designed hardware using PIC33FJ128MC802 micro-controller. Priority based task scheduled and switching carried with support for semaphores, yielding and system timers. Controller functions involved task scheduler based on priorities and kernel functions like yield, sleep, wait and signal for semaphore, and destroyProcess for resource handling.

      • Implementation of an Inverted Pendulum

        Designed and developed a self-balancer robot on the theory of Inverted Pendulum in Embedded C using PIC 33FJ128MC802. ZigBee module acted as a User Interface module to control the robot. Accelerometer and gyro-sensor are integrated using timer and PWM peripherals of the microcontroller and support the measurement of the orientation for PID controller to regulate the motor PWM.

      • PLC projects

        Flow Measurement:
        Designed a water flow measurement tool which will be able to indicate the volume of water flown in milliliters (mL) through a flow meter using Relay Ladder Logic.
        Data Analysis carried out for calibrating the design keeping flow rate constant.

        Temperature Measurement:
        Design and develop a temperature measurement tool that can validate the temperature in the range of 0°C to 90°C using a thermistor; a temperature sensor and IP3416 PLC.
        Data analysis is carried out on the resistance change with respect to the temperature and ladder logic is formulated to record the thermistor output and display the results in degree Celsius and Fahrenheit.

        Traffic Light System Design:
        Design a traffic light controller for a 4-way intersection based on pre-set time periods and flow of incoming traffic.

      • DMX512-Protocol Communication

        Achieved DMX512-A protocol communication between the master controller and other devices on a RS-485 communication bus with Embedded C and assembly level programming. (mainly used for intelligent lights – special effects or stage lighting)
        Designed controller and device circuitry for a timing intensive asynchronous communication interface using PIC33FJ128MC802 16-bit microcontroller.
        The master controller accepts user commands from PC RS-232 interface and controls other PIC devices.

      • SDRAM Controller Design

      • Polarization effects in MIMO system

        The project dealt with increasing the signal to noise ratio using single orthogonal polarity rather than two separate large antennas with QPSK technology over Raleigh fading channel

        Development carried out in MATLAB.
